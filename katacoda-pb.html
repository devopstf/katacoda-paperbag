Katacoda

“Interactive Learning and Training Platform for Software Engineers.”

Qué es

Una plataforma para montar de una manera sencilla un 'Quick Start' guiado sobre muchas herramientas y tecnologías relativamente populares utilizando para su consumo solo un navegador web.

Para qué sirve

Para montarte y mantener pequeños laboratorios interactivos y talleres de iniciación...

Elementos en un escenario

Pasos (steps.md) guiados y documentados.

Un fichero index.json que define todo el escenario.

Un entorno identificado con una imagen Docker.

Un layout que será nuestra UI.

Ficheros para cargar o ejecutar en el entorno

Por dónde empiezo

  1. Me hago un perfil (GitHub, LinkedIN, G+, Twitter)
  2. Creo un repo en GitHub y le asocio un WebHook (automatizado)
  3. Me bajo la CLI de Katacoda (v0.0.2)
  4. Creo mi primer escenario y lo subo a GitHub.

$ katacoda create scenario
$ katacoda push
